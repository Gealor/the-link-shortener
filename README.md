# Тестирование
Для тестирования у вас должен быть запущен Docker.
Чтобы запустить тесты введите команду 
```
pytest tests
```
Если хотите получить отчет о покрытии, то введите
```
pytest --cov=src tests
```

# Инструкция по локальному развертыванию сервиса
1. В /src/ создать .env, соответствующие примеры вы можете найти в /src/
2. Выполнить docker compose up --build

# Немного про CORS
CORS (Cross Origin Resource Sharing) - технология в веб-разработке, которая позволяет предоставлять доступ к ресурсу (API) с других доменов, отличного от того, на котором расположен API.
