# Цель проекта
Небольшой проект сокращателя ссылок, вдохновлено и сделано на примере видео Артема Шумейко https://www.youtube.com/watch?v=qe--uV1qJMI

# Что дальше?
1. Обернуть весь проект в docker-compose файл и поднять базу данных в нем.
1. Подключить базу данных Postgres и заносить туда соответствия между шорткатом и полной ссылкой. Вместе с этим реализовать слой Репозиторий.
2. Реализовать эндпоинт /{code}, который получает по шорткату полный, оригинальный URL.
3. Добавить тесты с помощью pytest (unit, интеграционные, системные) с помощью pytest. Добиться покрытия 80%.
